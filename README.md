# SenStyle - Pages légales

Ce dépôt contient les pages légales nécessaires pour valider l’application TikTok Developer de SenStyle.

## 📜 Pages incluses
- [Politique de Confidentialité](https://tonpseudo.github.io/senstyle-site/privacy-policy.html)
- [Conditions d’Utilisation](https://tonpseudo.github.io/senstyle-site/terms-of-service.html)

## 🚀 Hébergement
Les pages sont hébergées gratuitement via **GitHub Pages**.  
Pour activer :
1. Aller dans **Settings** du dépôt.
2. Section **Pages**.
3. Choisir la branche `main` et le dossier `/root`.
4. Sauvegarder.

## ✨ Objectif
Ces pages assurent la conformité de l’application avec les règles TikTok Developer et garantissent la transparence vis-à-vis des utilisateurs.
